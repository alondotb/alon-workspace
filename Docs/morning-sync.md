# Morning Sync

Just type **"morning sync"** or **"sync"** when starting a session. Claude will automatically:

1. Pull the VRT Tracker from Notion (all items not Done)
2. Show a dashboard grouped by Plane → Goal → Project → Task with progress %
3. Flag overdue or blocked items
4. Ask what's changed
5. Update Notion with your answers

This is built into CLAUDE.md — no copy-pasting needed.
