Design a UI for "Virtual Techies" (VRT) — a self-paced Python coding education platform for kids aged 10-14. Hebrew-first (RTL layout throughout), light theme. Kids learn through a 2D campaign node tree with "quests" (video-based lessons). No teacher needed. Game-quality UI — think Duolingo meets a 2D coding adventure.

Design principles:
1. Game, not school — every interaction feels like gameplay. Confetti on correct answers, fanfare on level-ups.
2. Zero adult help — a 10-year-old navigates independently. Large touch targets (44px+), clear icons, simple language.
3. Speed — instant feedback, max 3s loads.
4. Hebrew RTL — all UI mirrored right-to-left. Code blocks always LTR.
5. Light theme — bright, vibrant, kid-safe colors on white/light grey.
6. Celebrate everything — confetti, floating +XP text, streak fire, level-up overlays.

Color palette:
- Background: White #FFFFFF / Light grey #FAFAFA
- Primary accent: Vibrant purple #6C3AED
- Success: Green #22C55E
- Warning: Amber #F59E0B
- Error: Red #EF4444
- XP/Level: Gold #EAB308
- Coins: Amber gold
- Text: Dark grey #1F2937

Typography:
- Sans-serif with Hebrew support (Inter or Rubik)
- Body 16px minimum, headings 24-36px (big for kids)
- Code: Fira Code 14px, always LTR

Visual style:
- White cards on light grey background, generous spacing, rounded corners 12-16px
- Cards: white #FFF, subtle shadow, 16-24px padding
- Buttons: rounded 10-12px, large 44px+ height, bold text, primary = purple
- Icons: simple line-style 24px, emoji used liberally for gamification (🔥⭐🪙🏆🎉💡🐛)
- Smooth micro-animations on everything
- Responsive: Desktop 1200px+, tablet 768px, mobile 375px
