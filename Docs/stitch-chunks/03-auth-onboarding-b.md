Context: VRT (Virtual Techies) — coding education platform for kids 10-14. Hebrew RTL, light theme, gamified, purple accent #6C3AED. Design these 5 onboarding screens:

Screen 1 — Reset Password /reset-password
Centered card. New password field, confirm password field (both with show/hide toggle), password strength indicator bar, purple "Save" button.

Screen 2 — Add Child Form /parent/add-child
Card form. Username input with live availability check — shows ✅ green checkmark when available, ❌ red X when taken, spinner while checking. Birth year dropdown (2011-2016). Password input (min 4 chars). Purple "Add Child" button.

Screen 3 — Avatar Selection /onboarding/avatar
Full-screen, fun layout. Big "Choose Your Character!" headline. Grid of 10 diverse cartoon avatar characters (colorful, friendly, different skin tones/styles — some with glasses, hats, headphones). Large preview panel on the right showing the selected character full-body with subtle idle animation. Selected character has purple border + checkmark. Purple "Confirm" button at bottom.

Screen 4 — Avatar Selection Success
Celebration screen. Confetti particles everywhere. Large selected character illustration center. "Great choice!" headline. "Ready to learn to code?" subtext. Big purple "Go to First Lesson!" button. Bright, exciting energy.

Screen 5 — Parent Account Settings /parent/settings
Clean settings page. Sections: Email (with verification badge), Password (change button), Daily Time Limit slider (30min to 6hrs, default 2hrs, with current value label), Notification toggles (weekly email report ON/OFF, trial ending ON/OFF). Red "Delete Account" text button at bottom.

Show all 5 screens in a grid layout.
